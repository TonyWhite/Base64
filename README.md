# Base64

C++ Source for Base64.

## Encoder Features
* Standard
  * symbols `+` and `/`
  * optional padding `=`
* URL safe
  * symbols `-` and `_`
  * optional padding `=`

## Dev Features

* Simple to use
* Simple to Compile
* Only C++ standards
* CPU & RAM Optimized
* Compatible with all versions of C++
